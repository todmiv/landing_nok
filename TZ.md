# Техническое задание (ТЗ)
## Лендинг пейдж "НОК в строительстве и архитектуре"

### 1. Общие сведения

#### 1.1. Наименование проекта
"НОК в строительстве и архитектуре" - лендинг пейдж для Центра оценки квалификации ООО "Свой дом"

#### 1.2. Заказчик
ООО "Свой дом" (Центр оценки квалификации - ЦОК)

#### 1.3. Исполнитель
Фрилансер/разработчик

#### 1.4. Сроки выполнения
- Начало работ: 01.12.2025
- Завершение: 31.12.2025
- Приемка: 05.01.2026

### 2. Назначение и цели создания сайта

#### 2.1. Назначение
Создание современного, адаптивного лендинг пейдж для привлечения клиентов на услуги независимой оценки квалификации (НОК) в строительстве и архитектуре.

#### 2.2. Цели
- Информирование потенциальных клиентов о процедуре НОК
- Привлечение заявок на консультации и прохождение НОК
- Повышение узнаваемости бренда ЦОК "Свой дом"
- Автоматизация сбора заявок через формы

#### 2.3. Целевая аудитория
- Архитекторы и проектировщики
- Инженеры-строители
- Специалисты по организации строительства
- Руководители строительных компаний
- Частные специалисты строительной отрасли

### 3. Требования к сайту

#### 3.1. Технические требования

##### 3.1.1. Адаптивность
- Полная адаптивность под все устройства (desktop, tablet, mobile)
- Поддержка разрешений от 320px до 1920px+
- Bootstrap 5 responsive grid system

##### 3.1.2. Браузерная совместимость
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Opera 76+

##### 3.1.3. Производительность
- Время загрузки страницы < 3 сек
- Размер страницы < 2 MB
- Оптимизация изображений WebP/AVIF
- Lazy loading для изображений

##### 3.1.4. SEO оптимизация
- Мета-теги для каждой страницы
- Семантическая верстка HTML5
- Структурированные данные Schema.org
- Адаптивные изображения с alt-текстами
- ЧПУ (человекопонятные URL)

#### 3.2. Функциональные требования

##### 3.2.1. Структура сайта
1. **Главная страница (/)**
   - Hero секция с призывом к действию
   - Описание услуги НОК
   - Этапы прохождения процедуры
   - Преимущества прохождения НОК
   - Стоимость услуг (прайс-лист)
   - Расписание экзаменов
   - Форма заявки
   - FAQ секция
   - Отзывы клиентов
   - Контактная информация

2. **Страница консультации (/consultation.html)**
   - Подробное описание консультационных услуг
   - Форма записи на консультацию
   - Контактная информация

##### 3.2.2. Интерактивные элементы
- Формы обратной связи с валидацией
- Калькулятор стоимости НОК
- Модальные окна для форм
- Плавная прокрутка по якорям
- Анимации при скролле (AOS)
- Анимированные счетчики статистики

##### 3.2.3. Формы и отправка данных
- Форма заявки на главной странице
- Форма записи на консультацию
- Валидация полей (обязательные поля, формат email, телефона)
- Отправка данных в Google Sheets через Apps Script
- Обратная связь пользователю (успех/ошибка)

#### 3.3. Дизайн и UI/UX

##### 3.3.1. Цветовая схема
- Основной цвет: #2563eb (синий)
- Акцентный цвет: #f97316 (оранжевый)
- Серый: #6b7280
- Темный: #1f2937
- Светлый: #f8fafc
- Белый: #ffffff

##### 3.3.2. Типографика
- Основной шрифт: Inter (sans-serif)
- Резервные шрифты: system-ui, -apple-system, sans-serif
- Размеры текста: 14px-48px responsive
- Line-height: 1.6
- Font-weight: 400, 600, 700, 800

##### 3.3.3. Графические элементы
- Иконки: FontAwesome 6.4.0
- Логотип: SVG формат
- Favicon: SVG и ICO форматы
- Строительная тематика (иконки зданий, касок, чертежей)

##### 3.3.4. Анимации и эффекты
- Плавные переходы (0.3s ease)
- Hover эффекты на интерактивных элементах
- Анимации при скролле
- Loading состояния для форм
